# PHP 7.3 + Apache for Laravel

## Prerequisites

Install Docker Desktop (aka Docker for Mac)

- `brew install docker`

Docker Hub Login

- [Signup to Docker Hub](https://hub.docker.com/signup)
- `docker login`
